# Мир танков на macOS

Данная сборка позволяет играть в Мир танков без виртуальной Windows на macOS в 1 клик с Apple silicon и Intel процессором с работаюбщим Леста гейм центром и МОСТом (менеджер модов от Лесты).

Вся информация по сборке и возможным ошибкам собрана на странице:

https://playmac.ru/page/mir-tankov-faq



Видео с геймплеем:

#### [YouTube](https://www.youtube.com/watch?v=7CJCI_9IcEo)

#### [Rutube](https://rutube.ru/video/de6cbdcc3d2ae8d2a3a0896d3e77643e)

Проект перешёл на донатно-подписочную основу.
Готовую сборку можно получить либо на проекте Playmac:

https://playmac.ru/game/mir-tankov

Либо на Boosty:

https://boosty.to/bmaxus

Если возникнут вопросы контакт можете найти в профиле гитхаба.
