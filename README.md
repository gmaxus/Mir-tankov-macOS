# Мир танков на macOS

Данная сборка позволяет играть в Мир танков без виртуальной Windows на macOS в 1 клик с Apple silicon и Intel процессором с работаюбщим Леста гейм центром и МОСТом (менеджер модов от Лесты).

Вся информация по сборке и возможным ошибкам собрана на странице:

https://playmac.ru/page/mir-tankov-faq
